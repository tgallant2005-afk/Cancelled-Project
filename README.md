# Cancelled-Project
Project I was working on during the 2025 summer, currently cancelled but might be worked on in the future

Original Idea was to make a terminal based U of M degree planner, one which could notify the user of any potential conflicts, as well as show all possible combinations of chosen classes based on available class times.
The program was also planned to be *somewhat* future-proofed, as one could either give the program class info (pre-requisites, time slots, etc.) or have it go to the university catalogue website and parse the information itself, which would be downloaded to a text file and stored with the program for future uses (along with having the option to write the current plan to a schedule and import it back in)
